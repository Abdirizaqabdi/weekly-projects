# Movie App

A movie dashboard where the user can search and explore movies

## User Stories

-   [ ] User can can see a list of movies.
-   [ ] User can search and filter to only see specific movies.

## Bonus features

-   [ ] Interactive Dashboard.
-   [ ] Pagination.
-   [ ] User can add movies.
